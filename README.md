A ideia aqui é começar pequeno, aprender o caminho com calma e deixar tudo bonitinho conforme o projeto crescer. 🚀

## 🌟 Objetivo

Este projeto serve como um laboratório leve para experimentar:

- 🧠 edição de arquivos com apoio do Codex;
- 🌱 controle de versão com Git;
- ☁️ sincronização com GitHub;
- 🔀 pull requests em vez de push direto na `main`;
- ✅ issues, reviews e checks no futuro.

## 🛠️ Como usar

Clone o repositório:

```bash
git clone https://github.com/007msn98/Codex-0.git
```

Entre na pasta:

```bash
cd Codex-0
```

Confira o estado do Git:

```bash
git status
```

## 🔐 Fluxo recomendado

Para manter a `main` protegida e limpinha:

1. Crie uma branch para cada mudança.
2. Faça commits pequenos e claros.
3. Abra um pull request.
4. Revise, ajuste e só depois faça merge.

Exemplo:

```bash
git switch -c codex/minha-mudanca
```
