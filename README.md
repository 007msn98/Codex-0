# Codex-0

Repositorio inicial para testar e organizar fluxos de trabalho com o Codex, Git e GitHub.

## Objetivo

Este projeto serve como um espaco simples para experimentar:

- edicao de arquivos com apoio do Codex;
- controle de versao com Git;
- sincronizacao com GitHub;
- abertura futura de issues, pull requests e checks.

## Status

Projeto em fase inicial. No momento, o repositorio contem apenas a base minima para evoluir os proximos testes.

## Como usar

Clone o repositorio:

```bash
git clone https://github.com/007msn98/Codex-0.git
```

Entre na pasta do projeto:

```bash
cd Codex-0
```

Confira o estado do Git:

```bash
git status
```
